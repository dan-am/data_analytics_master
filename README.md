A comprehensive data analytics project implementing the complete Data Science Life Cycle for patient segmentation analysis.

**Location:** [`patient_segmentation_project/`](patient_segmentation_project/)

**Dataset:** [Patient Segmentation Data from Kaggle](https://www.kaggle.com/datasets/nudratabbas/patient-segmentation-data)

**Features:**
- Complete 8-phase Data Science Life Cycle implementation
- Ready-to-use scripts for data acquisition, cleaning, and validation
- Comprehensive documentation for each phase
- Example code for modeling and evaluation
- Deployment guidelines and inference scripts
- Professional project structure with best practices

**Quick Start:**
```bash
cd patient_segmentation_project
pip install -r requirements.txt
python 2_data_acquisition/download_data.py
```
